Radar Chart

 

12 vertex Radar graph (SVG) PowerApps component ready to be used.

Import it to your project and use following variables :

  • LabelA - thru LabelL, to give name to each vertex
  • ValueA - thru ValueL, to enter the values.
  • minValue - Minimum allowed value
  • maxValue - Maximum allowed value
  • strokecolor - colour of the shadow zone.
